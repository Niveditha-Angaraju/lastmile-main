# LastMile Frontend

Interactive web frontend for the LastMile ride matching system.

## Setup

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

The frontend will be available at http://localhost:3000

## Build

```bash
npm run build
```

## Backend API

The frontend expects a backend API server running on port 8080. See `backend/` directory for the API server implementation.
